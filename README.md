Prevent - Игра про офисного работника, который застрял во временной петле на научной станции по изучению демонов и пытается предотвратить их массовый побег.

Игра создавалась для двухнедельного геймджема csu fall semester 2025. 

В игре реализованы:
- Простая система диалогов
- Простая система анимаций
- Подбор и использование предметов
- Механика стрельбы
- Простой алгоритм поведения противников
- Обратный отсчет до поражения с перезагрузкой уровня

Планы на доработку:
- Улучшить систему диалогов
- Добавить новые пути решения проблемы
